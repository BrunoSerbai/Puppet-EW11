# Puppet-EW11

Automated testing and diagnostics tool for Elfin-EW11 modules using Puppeteer.
Functions

    Automated Auth: Handles Basic Access Authentication.

    Diagnostics: Scrapes system status (RAM, RSSI, Network) and captures element-specific screenshots.

    Remote Reset: Executes hardware reboot and handles JavaScript confirmation dialogs.

    Error Handling: Manages connection drops and TargetCloseErrors during device reboot cycles.
